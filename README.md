# Gram Discord Presence

Gram Discord Presence adds Discord Rich Presence support to **Gram** using an LSP.

## Requirements

- Rust
- rustup (recommended)

## Installation

Because Gram doesn't have an extension registry.

Open the Command Palette and run:

`Gram: Install Extension from URL`

`https://github.com/playfairs/gram-discord-presence`

### Development Installation

1. Clone this repository.
2. Run `Gram: Install Extension from folder`.
3. Select the cloned directory.
4. Reload Gram.
